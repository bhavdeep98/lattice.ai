# IP Transfer Template

## When You Register Your Company

### 1. Update LICENSE file:
```
Licensor: [Company Name]
Licensed Work: Lattice AWS CDK
The Licensed Work is (c) 2025 [Company Name]
```

### 2. Update README.md:
```
For commercial licensing inquiries, please contact licensing@[company-domain].com
```

### 3. Update COMMERCIAL-LICENSE.md:
```
Email: licensing@[company-domain].com
Website: https://[company-domain].com
```

### 4. Add copyright assignment:
Create a simple document stating:
"I, [Your Name], hereby assign all rights, title, and interest in the Lattice AWS CDK software to [Company Name], effective [Date]."

### 5. File with USPTO (Optional but Recommended):
- Trademark "Lattice" for software/cloud services
- Consider provisional patent for unique algorithms

## Legal Notes
- Copyright exists from creation (no registration required)
- BSL 1.1 is legally enforceable
- Personal ownership transfers cleanly to company
- Keep development logs as proof of creation
